A map of every page on the new site and what lives on each one. This pairs with the full content draft. Review the hierarchy, mark anything you want moved, renamed, added, or cut, and approve so we can lock it and start building.

Version 1.0 · June 3, 2026 · For board review & approval

How to read this: Each box is one page. The tags show what is changing from the current site. Bullet points are the gist of what goes on each page, not final copy (final copy lives in the companion content doc). The top row is the main menu; below it are the supporting pages reached from the footer or from within other pages.
