# Benson Theatre — Proposed Site Structure

Client-facing visual sitemap for the Benson Theatre website rebuild, served via GitHub Pages.

**Live URL:** https://ericdowns.github.io/benson-theatre-sitemap/

## Updating

1. Edit `index.html`.
2. Bump the version badge in the header (`<span class="ver">Version X.Y</span>`).
3. Add a new `<div class="row">` to the **Version history** block near the bottom describing what changed.
4. Commit and push to `main` — GitHub Pages redeploys automatically in ~1 minute.
